飲み会で遊べるボードゲーム企画 10案

飲む量で盛り上げるのではなく、乾杯・水・つまみ・注文・会話・席順をゲームの中心に置く。宅飲み、バー、ゲーム会で安全に遊べる企画として、上位10案を試作に進めやすい粒度まで整理しました。

飲酒は任意 ノンアル・水で代替可 カード中心で試作可能 対象: 大人の飲み会/ゲーム会

企画の前提

全案共通で、一気飲み・飲酒強制・罰ゲーム飲酒は入れません。飲み物は演出と選択肢であり、ゲーム上の処理はカード、トークン、宣言で完結します。

3-6人 初回ターゲット。飲み会の卓で自然に始めやすい人数。
15-45分 軽量作は15分、経営/協力寄りは30分前後を想定。
小箱 カード、チップ、席札、つまみトークンで初回試作可能。
安全設計 飲む代わりに、水・ノンアル・お通し・乾杯札で処理。

10案ナビ

クリックで各企画へ移動できます。まず作るなら、会話推理の「乾杯アリバイ」か、小箱会話ゲームの「お通し心理戦」が試作しやすいです。

企画詳細

各案は、最低限のコンポーネントと1ラウンドの動きが見えるように整理しています。紙・付箋・トランプ・コインで初回テストに入れる粒度です。

01

酔わない酒場戦略

客の満足度と安全管理を両立する、少しブラックで優しいバー運営ゲーム。

Players2-5人
Time30-45分
Type半協力・経営
Prototypeカード80枚 + トークン

遊びの核

プレイヤーは共同バーのスタッフ。注文を取って売上を伸ばすほど、客の疲れ・空腹・水分不足が増える。売上、評判、安全の3指標を全部守りながら、自分の担当目標も達成する。

基本の流れ

  1. 客カードを公開し、注文・性格・注意状態を確認する。
  2. 各プレイヤーが「提供」「水を出す」「つまみを勧める」「休憩席へ案内」「会話で落ち着かせる」から同時選択する。
  3. 注文が成功すると売上が増えるが、ケアが不足すると評判や安全ゲージが下がる。
  4. 閉店時に店が潰れず、個人目標も満たした人が勝利候補になる。

コンポーネント

客カード 注文カード 水トークン つまみトークン 評判/安全/売上ボード
サンプルカード 客「勢いだけの新人」: 注文成功で売上+2。ただし水が出ていないと安全-1。つまみを出すと評判+1。

飲みの絡み方

飲むことではなく、水・つまみ・休憩を資源化する。プレイヤー自身は実飲しなくてよく、卓上のケア判断がゲームになる。

磨きどころ

説教臭くならないように、客のクセとイベントをコミカルにする。売上を狙う人と安全を守る人の温度差が笑いになる設計が大事。

安全ルール: ゲーム中の「提供」はカード処理。実際の飲酒量には一切連動させない。
02

バー救済24時

閉店寸前のバーを、全員で一晩だけ立て直す協力ゲーム。

Players3-6人
Time25-40分
Type完全協力・危機管理
Prototypeイベントカード + 役職カード

遊びの核

プレイヤーはマスター、料理担当、ホール、常連、助っ人などの役職を持つ。トラブルが毎ラウンド発生し、乾杯・水提供・つまみ補充・会計・片付けを分担して店の存続ゲージを守る。

基本の流れ

  1. トラブルカードをめくる。「団体客」「氷切れ」「常連の相談」「終電間近」など。
  2. 全員が手札から1枚を伏せ、同時公開する。
  3. 役職能力とカード効果で、売上・雰囲気・体力・安全ゲージを調整する。
  4. 24時までに借金を返し、常連ゲージを一定以上にすれば勝利。

コンポーネント

役職カード トラブルカード 対応カード 店ゲージ 時間トラック
サンプルカード トラブル「謎の二次会客」: 売上+3のチャンス。対応に水かつまみがない場合、雰囲気-2。

飲みの絡み方

乾杯は全員の士気回復、水とつまみは安全維持。飲み会の所作が協力アクションとして自然に入る。

磨きどころ

協力ゲーム特有の指示役問題を避けるため、同時選択と個別の隠し小目標を入れると良い。

安全ルール: 乾杯はグラスを掲げるだけでも成立。水やノンアルを標準として扱う。
03

乾杯アリバイ

乾杯した相手とだけ秘密情報を交換できる、飲み会推理ゲーム。

Players4-8人
Time15-25分
Type会話推理・正体隠匿
Prototype秘密カード + 乾杯札

遊びの核

ある夜、誰かが「最後の一杯」をすり替えた。各プレイヤーは断片的な記憶を持つ。ラウンドごとに1人を選んで乾杯し、その相手とだけ秘密情報を交換できる。誰と情報交換するかがそのまま疑惑になる。

基本の流れ

  1. 全員に役割カードと記憶カードを配る。犯人役だけ目的が違う。
  2. 各ラウンド、全員が乾杯したい相手を同時に指名する。
  3. 相互指名なら密談成功。片思い指名なら公開ヒントだけ得る。
  4. 3ラウンド後に討論し、犯人とすり替え理由を投票する。

コンポーネント

役割カード 記憶カード 乾杯指名札 疑惑トークン 事件シート
サンプルカード 記憶「22:18、青いグラスを持つ人が席を外した。ただし相手が誰かは覚えていない。」

飲みの絡み方

乾杯相手の選択が情報ルートになる。飲む必要はなく、乾杯札・グラスを掲げる・水で乾杯で処理できる。

磨きどころ

事件の種類を複数用意するとリプレイ性が出る。最初は「すり替え」「消えた会計」「謎の裏メニュー」など軽めが合う。

安全ルール: 乾杯は情報交換の合図。実際に飲むかどうかは完全任意。
04

乾杯同盟

店を繁盛させるために協力する。でも、最後に得する相手は選びたい。

Players3-6人
Time25-35分
Type半協力・交渉
Prototype同盟カード + 店ボード

遊びの核

全員で店の売上を伸ばさないと全員敗北。ただし各自には「甘口メニューを一番売る」「特定プレイヤーより評判を取る」などの個人目標がある。乾杯すると共同アクションが強化されるが、相手にも利益が入る。

基本の流れ

  1. 今夜の客層と流行ドリンクを公開する。
  2. プレイヤーは手札から店アクションを選び、必要なら誰かと乾杯同盟を結ぶ。
  3. 同盟した2人は効果が上がるが、ボーナス配分を交渉する。
  4. 閉店時、店が基準点を超えたうえで個人点を比べる。

コンポーネント

店アクションカード 個人目標カード 乾杯同盟札 売上/評判トラック
サンプルカード 同盟「一杯目キャンペーン」: 2人で出すと売上+3。どちらか片方が評判+1を得る。

飲みの絡み方

乾杯が協力の契約になる。グラスを合わせる、札を置く、合言葉を言うなど、場に合わせて演出できる。

磨きどころ

半協力なので、裏切りが強すぎると場が荒れる。裏切りより「誰を得させるか」の悩みに寄せると飲み会向き。

安全ルール: 乾杯は同盟宣言であり、飲酒行動ではない。ノンアル乾杯を標準ルールに入れる。
05

秘密のペアリング競売

料理とドリンクの最高の組み合わせを、競りと読み合いで作る。

Players2-5人
Time20-30分
Type競り・セット収集
Prototype料理/ドリンクカード各40枚

遊びの核

場に並ぶ料理カードとドリンクカードを競り落とし、自分だけのコースを作る。カードには「甘い」「酸味」「香ばしい」「軽い」「濃い」などのタグがあり、相性が良いほど得点が伸びる。

基本の流れ

  1. 料理3枚、ドリンク3枚を公開する。
  2. プレイヤーはチップで競り、1枚ずつ獲得する。
  3. 獲得後、手元の料理とドリンクを組み合わせてペアリングを作る。
  4. 秘密の好みカードと公開トレンドで得点する。

コンポーネント

料理カード ドリンクカード 好みカード 競りチップ 感想カード
サンプルカード 料理「焦がしチーズ」: 香ばしい/濃い。ドリンク「柑橘ソーダ」: 酸味/軽い。相反タグで意外性+2。

飲みの絡み方

実際の味見は任意。飲み物やつまみを用意できる場では感想カードでボーナス、用意できない場ではタグだけで遊べる。

磨きどころ

食べ物/飲み物の名前が魅力を左右する。コンビニつまみ版、居酒屋版、クラフトドリンク版など展開しやすい。

安全ルール: 味見は任意の演出。ゲーム上の得点はカードタグで完結する。
06

つまみ総選挙

今夜いちばん愛されるおつまみを、演説と裏工作で決める。

Players3-8人
Time15-25分
Type投票・大喜利・軽交渉
Prototype候補カード + 投票札

遊びの核

各プレイヤーは推しつまみ陣営の選対担当。枝豆、唐揚げ、チーズ、ポテチ、冷奴などの候補を、ドリンク属性や場の空気に合わせて推していく。勝つのは単純な人気ではなく、裏公約と相性ボーナスを読んだ陣営。

基本の流れ

  1. 今夜のドリンク属性を公開する。例: 泡、甘口、辛口、ノンアル。
  2. 各自が推しつまみカードを1枚選ぶ。
  3. 演説カードや裏公約カードで票を動かす。
  4. 公開投票と隠し投票を合算し、相性ボーナスを加えて勝者を決める。

コンポーネント

つまみ候補カード ドリンク属性カード 演説カード 裏公約カード 投票札
サンプルカード 裏公約「締めまで責任を持ちます」: 最終ラウンドで票+2。ただし途中で最下位なら無効。

飲みの絡み方

ドリンク属性が選挙情勢を変える。実物のつまみがあると盛り上がるが、カードだけでも成立する。

磨きどころ

候補つまみのキャラ付けが命。真面目な選挙用語とくだらないつまみ論争の落差を強くする。

安全ルール: 主役はつまみ。飲酒ではなく相性と投票で盛り上げる。
07

マスターの席札

客の相性を読み、最高のカウンター席を作る配置パズル。

Players1-4人
Time20-30分
Type配置パズル・接客
Prototype客カード + 席ボード

遊びの核

プレイヤーはバーのマスター。客カードには好みの飲み属性、隣に座りたいタイプ、避けたいタイプが書かれている。席順を調整して、店内の空気と売上を最大化する。

基本の流れ

  1. 新規客カードを3枚公開する。
  2. 手番プレイヤーは1人を席に案内し、必要なら既存客を1席だけ動かす。
  3. 隣接条件、飲み属性、会話相性で得点する。
  4. 満席になったらイベントを解決し、最終評判を比べる。

コンポーネント

カウンターボード 客カード 飲み属性チップ 空気イベントカード
サンプルカード 客「聞き上手の常連」: 甘口の隣で+1。愚痴りたい客の隣で+2。騒がしい客の隣で-1。

飲みの絡み方

甘口、辛口、泡、香り、ノンアルなどの属性が席配置に影響する。飲む量は不要で、飲み物の個性だけ使う。

磨きどころ

地味になりやすいので、客キャラの名前と効果を尖らせる。ソロ対応もできるので試作検証がしやすい。

安全ルール: 飲み属性はカード効果。実際の飲み物を用意しなくても遊べる。
08

お通し心理戦

お通しを渡して質問権を得る、軽く深い飲み会会話ゲーム。

Players3-7人
Time10-20分
Type会話・心理戦・小箱
Prototype質問カード + お通しトークン

遊びの核

プレイヤーはお通しトークンを誰かに渡すことで質問権を得る。ただし質問には「軽い」「深い」「変な」の3段階があり、相手は答えるか、別のお通しを返してかわすか選べる。本音を引き出すほど得点になるが、踏み込みすぎると空気ポイントを失う。

基本の流れ

  1. 質問カードを3枚公開する。
  2. 手番プレイヤーはお通しトークンを1つ渡して質問する。
  3. 答えた人は共感札、回避した人はお通し返しで処理する。
  4. 最後に「一番いい話を引き出した人」と「一番場を守った人」が得点する。

コンポーネント

質問カード お通しトークン 共感札 回避札 空気ポイント
サンプルカード 質問「最近ちょっとだけ自分を褒めたいことは？」軽い/共感+1。深掘りすると追加点。

飲みの絡み方

飲む代わりにお通しを渡す。つまみや小皿のイメージで、会話のきっかけを自然に作れる。

磨きどころ

質問の温度管理が最重要。初対面でも使える軽い質問と、仲の良い場で刺さる質問を分ける。

安全ルール: 答えたくない質問はいつでもパス可能。パスしても不利になりすぎない。
09

バー・ミラージュ

飲み物の味や雰囲気から、幻のバーを全員で作る物語ゲーム。

Players3-6人
Time20-35分
Type物語生成・SNS映え
Prototype噂カード + 店づくりカード

遊びの核

各ラウンド、飲み物やカードから連想して「幻のバー」の噂を追加する。店名、看板メニュー、常連、禁止事項、夜だけ起きる現象が積み上がり、最後に最も魅力的なバーとして語れるかを競う。

基本の流れ

  1. テーマカードを公開する。例: 雨、忘れ物、秘密、終電。
  2. プレイヤーは手札から噂カードを出し、店の設定を1つ語る。
  3. 他プレイヤーは共鳴札で「行きたい」「怪しい」「名物っぽい」を投票する。
  4. 完成したバーの短い紹介文を作り、最多共鳴の人が勝つ。

コンポーネント

テーマカード 噂カード 常連カード 共鳴札 店名シート
サンプルカード 噂「その店では、最後に頼んだ一杯の名前を翌朝まで思い出せない。」

飲みの絡み方

味、色、氷、香り、泡を物語の素材にする。ノンアルでも「この色ならどんな店か」で十分成立する。

磨きどころ

勝敗を緩くしすぎないために、共鳴札とテーマ達成点を入れる。写真に撮りたくなる店シートも相性がいい。

安全ルール: 飲み物は創作の素材。実飲なしでも色カードや香りカードで遊べる。
10

乾杯写真館

グラス配置とポーズで、最高の乾杯写真を作る構図ゲーム。

Players2-8人
Time10-20分
Typeアクション少なめ・写真/構図
Prototype構図カード + 得点札

遊びの核

お題カードに合わせて、グラス、手、つまみ、席札を配置し、乾杯写真の構図を作る。実際に撮影してもいいし、テーブル上の配置だけで判定してもいい。映えるだけでなく、制約パズルとして遊ぶ。

基本の流れ

  1. お題カードを公開する。例: 三角形、秘密の同盟、最後の一杯、全員主役。
  2. 各チームが制限時間内に構図を作る。
  3. 得点札で「構図」「物語」「再現難度」「楽しそう」を評価する。
  4. 3ラウンドの合計で勝敗を決める。

コンポーネント

構図お題カード 評価札 小物トークン 制限時間カード
サンプルカード お題「秘密の同盟」: 2人だけが同じ角度でグラスを傾ける。他の人は気づいていない構図にする。

飲みの絡み方

飲むのではなく、グラスやつまみの見せ方をゲーム化する。SNS投稿企画やイベント卓に向いている。

磨きどころ

写真が苦手な人でも遊べるよう、撮影なしの「卓上構図判定」ルールを標準にする。

安全ルール: グラスは空でも水でもよい。撮影や投稿は全員の同意がある場合だけ。

次に作るなら

最短で試作するなら、カード枚数が少なくて場の反応を見やすい案から始めるのが良いです。最初のテストは完成度より「1回遊んで、もう1回やりたいか」を見るのが目的です。

最速試作 乾杯アリバイ。役割カード、記憶カード、乾杯札だけで初回テスト可能。
販売しやすい小箱 お通し心理戦。質問カードの質で商品価値を作りやすい。
大きく育つ企画 酔わない酒場戦略。拡張、イベント、客キャラ展開がしやすい。
