01

乾杯裁判

グラスを証拠品にして、誰が一番しょうもない罪かを裁く法廷バカゲー。

Players3-7人
Time15-25分
Type裁判・大喜利・投票
Baka罪名が全部くだらない
Testカード40枚から可

遊びの核

毎ラウンド1人が被告になる。罪状カードを引き、検察・弁護・証人が「グラスの高さ」「乾杯の角度」「つまみの位置」を証拠としてこじつける。最後に裁判長が判決を出し、もっともしょうもない理屈を通した人が得点する。

1ラウンドの流れ

  1. 罪状カードを公開する。
  2. 被告は「やってません」と無駄に堂々と宣言する。
  3. 検察と弁護が証拠カードを1枚ずつ出して主張する。
  4. 全員投票で「有罪」「無罪」「意味不明賞」を決める。

コンポーネント

罪状カード 証拠カード 判決札 異議ありトークン 乾杯ベル
サンプルカード 罪状「氷が丸すぎる罪」: 被告はグラスの中に隠された美意識について弁明する。検察は丸さの危険性を主張する。

飲みの絡み方

乾杯は判決の拍子木。飲む必要はなく、空グラスの音や乾杯ポーズで裁判が進む。

磨きどころ

罰として飲ませないこと。判決は「乾杯コール」「証拠トークン没収」「次の裁判長」などに寄せる。

安全設計: 有罪でも飲酒ペナルティなし。水・空グラス・拍手で完全代替。
02

注文してません

誰も頼んでない謎注文が勝手に届き、全員で押し付けたり処理したりする運ゲー。

Players2-6人
Time10-20分
Typeカード・カオス・押し付け
Baka注文名と効果が無関係
Test注文カードだけで可

遊びの核

山札から「頼んでない注文」が届く。プレイヤーは受け取る、隣に回す、店員を呼ぶ、記憶を捏造するなどのカードで処理する。手元に謎注文がたまりすぎると会計圧が上がる。

1ラウンドの流れ

  1. 親が注文カードを1枚めくる。
  2. 全員が「頼んだ/頼んでない/見た気がする」を同時宣言する。
  3. 処理カードで注文を押し付けるか、会計トークンとして引き取る。
  4. 3枚以上抱えた人は「店員さん確認フェイズ」に入る。

コンポーネント

謎注文カード 処理カード 会計圧トークン 店員さん札
サンプルカード 注文「ウーロン茶」: 全員は手札を反時計回りに回す。その後、回ってきた手札に小声で謝罪する。

飲みの絡み方

飲み物名がカード効果のトリガーになる。実物の飲み物は不要で、架空注文だけで遊べる。

磨きどころ

初回は1行効果だけにする。効果文が長いとバカさより処理負荷が勝ってしまう。

安全設計: 「注文」はカード上の概念。実際の注文や飲酒とは連動しない。
03

店長、今日のおすすめ変です

終わっている名前のドリンクを、なぜか真剣に売り込む即興セールスゲーム。

Players3-8人
Time15-25分
Type大喜利・プレゼン
Baka商品名が全部ひどい
Test単語カードで可

遊びの核

プレイヤーは日替わり店長。ランダムに引いた「味」「効能」「売り文句」を組み合わせて、今日のおすすめドリンクを作り、30秒で売り込む。客役は最も注文したくないのに気になる一杯へ投票する。

1ラウンドの流れ

  1. 味カード、効能カード、名前カードを1枚ずつ引く。
  2. 30秒で商品名と売り文句を作る。
  3. 全員が店長口調でプレゼンする。
  4. 「頼みたくないけど忘れられない」票を集めた人が勝つ。

コンポーネント

味カード 効能カード 名前カード 注文票 店長バッジ
サンプルカード 名前「初恋の残業ハイボール」+ 効能「明日の予定を少しだけ他人事にする」+ 味「反省している桃」。

飲みの絡み方

架空ドリンクの注文ごっこ。乾杯は「ご注文ありがとうございます」の合図で、実飲は不要。

磨きどころ

即興が苦手な人向けにテンプレを入れる。「当店では」「こちらは」「余韻に」などの補助語があると遊びやすい。

安全設計: 架空メニュー限定。実際に混ぜる、飲ませる、注文する必要なし。
04

乾杯禁止区域

乾杯したいのに条例が邪魔をする。合法乾杯スポットを探す配置バカゲー。

Players2-5人
Time20-30分
Type配置・謎ルール
Baka斜め45度が文化財
Test紙マップ + グラスコマ

遊びの核

テーブル上に街マップを置き、グラスコマを合法乾杯スポットへ移動させる。ただし毎ラウンド「北半球では持てない」「赤いコースター上だけ合法」などの条例が出る。

1ラウンドの流れ

  1. 条例カードを2枚公開する。
  2. 各プレイヤーはグラスコマを1-2マス動かす。
  3. 合法条件を満たす相手とだけ乾杯宣言できる。
  4. 違法乾杯は減点ではなく「謎の行政指導」カードを受け取る。

コンポーネント

街マップ 条例カード グラスコマ 行政指導カード 合法乾杯札
サンプルカード 条例「斜め45度の乾杯は文化財」: グラスコマが斜め向きで接している時だけ乾杯できる。

飲みの絡み方

グラスの位置・向き・距離がルールの中心。飲むのではなく、合法な乾杯ポーズを作る。

磨きどころ

場に出る条例は常時2-3枚まで。増やしすぎると笑いより確認作業が増える。

安全設計: 実物グラスではなく、空カップや紙コマを標準にする。
05

乾杯官房長官

しょうもない出来事を、国の重大発表みたいに読み上げる公式声明ゲーム。

Players3-8人
Time10-20分
Type声明・採決・大喜利
Baka割り箸が歴史的偉業
Testお題カード中心

遊びの核

お題カードに書かれた小さな出来事を、官房長官っぽく正式発表する。全員は質問記者になって、どうでもいい追及をする。採用された声明で乾杯コールが決まる。

1ラウンドの流れ

  1. 出来事カードを公開する。
  2. 発表者は声明テンプレに沿って30秒で発表する。
  3. 記者役が1人1問だけ質問する。
  4. 最も公式っぽかった人に「閣議決定」トークン。

コンポーネント

出来事カード 声明テンプレ 記者質問カード 閣議決定トークン
サンプルカード 出来事「割り箸がきれいに割れた」: 国民生活への影響、今後の対応、関係各所への感謝を必ず述べる。

飲みの絡み方

採用声明がそのまま乾杯文になる。飲む必要はなく「以上をもちまして乾杯」と唱えるだけでも成立。

磨きどころ

大喜利一本にしないため、テンプレ穴埋めと質問カードで誰でもそれっぽく話せるようにする。

安全設計: 乾杯は発表の締め。水、空グラス、拍手で代替可能。
06

居酒屋怪獣襲来

卓上の居酒屋街を、酔ってない怪獣たちが理不尽に踏み荒らす盤面バカゲー。

Players2-5人
Time20-35分
Type卓上破壊・協力/対戦
Baka唐揚げ発電所を守る
Test紙マップ + 小物

遊びの核

グラスや皿をビル、つまみを施設に見立て、怪獣コマの進路を変えながら居酒屋街を守る。乾杯すると怪獣が音に反応して方向転換するが、守りたい施設も巻き込まれる。

1ラウンドの流れ

  1. 怪獣カードをめくり、移動パターンを確認する。
  2. プレイヤーはグラスビルやつまみ施設を配置する。
  3. 任意で乾杯音を使い、怪獣の向きを変える。
  4. 守れた施設と壊れた施設で得点や被害を計算する。

コンポーネント

怪獣カード 居酒屋街マップ 施設トークン 進路カード 被害チップ
サンプルカード 施設「唐揚げ発電所」: 壊れると全員のテンション-2。守ると次の乾杯音が2回使える。

飲みの絡み方

グラスはビル、乾杯音は怪獣誘導。飲み物の量ではなく、卓上の見立てで笑わせる。

磨きどころ

固有怪獣名は避け、オリジナル怪獣にする。グラス転倒防止のため空カップ・紙コップ推奨。

安全設計: 実物の液体入りグラスを倒す遊びにしない。空カップと紙トークンで標準化。
07

乾杯したら世界が変わった

乾杯するたびに勝利条件が変わり、さっきまで勝っていた人が急に迷子になる短期決戦。

Players3-6人
Time15-25分
Type勝利条件変更・カオス
Baka勝ってたのに負け担当
Test目的カードだけで可

遊びの核

各自が目的カードを持って点を稼ぐ。ただし乾杯カードが切られるたびに、世界カードが更新され、勝利条件や得点方法が変わる。「手札が少ないほど偉い」「一番負けている人が世界の希望」など理不尽だが短い。

1ラウンドの流れ

  1. 全員が目的カードを1枚受け取る。
  2. 手番でカードを出して得点やトークンを集める。
  3. 誰かが乾杯カードを出すと世界カードを更新する。
  4. 3回目の世界更新後、現在の勝利条件で勝者を決める。

コンポーネント

目的カード 世界カード 乾杯カード 得点トークン
サンプルカード 世界「今から全員、昨日の自分」: 直前ラウンドで取った得点だけが有効。多く持っていたトークンは思い出になる。

飲みの絡み方

乾杯が世界変更ボタン。空グラスを合わせるだけで発動するので、飲酒量には依存しない。

磨きどころ

勝利条件は短く明快に。変化が多すぎると疲れるので、ゲーム中3回程度がちょうどいい。

安全設計: 乾杯はカード効果のスイッチ。飲む必要なし。
08

メニューにないやつ

存在しない飲み物を注文し、その効能を全員で信じる架空メニュー生成ゲーム。

Players3-8人
Time10-20分
Type架空メニュー・即興
Baka概念サワーが普通に通る
Test単語カードで可

遊びの核

プレイヤーは存在しないドリンクを注文する客。名前カードと効能カードを組み合わせて「メニューにないやつ」を作り、店員役に通す。店員役はその飲み物が出せる理由か、出せない理由を即興で返す。

1ラウンドの流れ

  1. 客は名前カード2枚を組み合わせて注文名を作る。
  2. 効能カードを1枚付けて「これください」と宣言する。
  3. 店員は提供、品切れ、裏メニュー化のどれかで返す。
  4. 全員が一番信じたい架空ドリンクへ投票する。

コンポーネント

名前カード 効能カード 店員リアクション札 裏メニュー票
サンプルカード 注文「概念サワー」: 飲むと、さっき言いかけた話を全員が知っていたことにできる。

飲みの絡み方

架空ドリンクを中心に会話が回る。実物は出さない。グラスを掲げるだけで「届いた」扱い。

磨きどころ

効能カテゴリを固定すると遊びやすい。例: 記憶、席順、手札、会計、テンション。

安全設計: 架空ドリンク専用。実際に混ぜる・飲む・飲ませる処理は禁止。
09

コースター大相撲

コースター力士を土俵に入れ、酒場横綱を決める安全な物理バカゲー。

Players2-6人
Time10-20分
Type物理・配置・トーナメント
Baka決まり手がおしぼり投げ
Test紙コースターで可

遊びの核

紙コースターを力士に見立て、テーブル土俵の上で押し合い、配置し合う。実際に強く弾くのではなく、カードで「押す」「かわす」「土俵際」「スポンサー席」などの効果を解決する。

1ラウンドの流れ

  1. 各自が力士コースターを1枚選ぶ。
  2. 土俵カードで初期位置を決める。
  3. 同時に技カードを出し、位置と向きを調整する。
  4. 土俵外に出たら負け。残った力士に懸賞トークン。

コンポーネント

力士コースター 技カード 土俵シート 懸賞トークン 決まり手カード
サンプルカード 決まり手「レモンしぼり」: 相手を1マス横へずらす。相手が土俵際なら追加で90度回転。

飲みの絡み方

グラス位置が土俵際やスポンサー席になる。飲む必要はなく、卓上の小物で相撲場を作る。

磨きどころ

液体入りグラスは使わない。紙コップ、空グラス、コースターだけで安全な物理感を出す。

安全設計: 強く弾く、こぼす、割れる可能性のある遊びは禁止。カード解決を中心にする。
10

二次会ドラゴン

「一杯だけだから」と誘ってくるドラゴンを、水と休憩とやんわり断りで帰す協力ゲーム。

Players3-6人
Time20-30分
Type協力・帰宅・あるある
Bakaドラゴンの口癖が一杯だけ
Testドラゴンカード + 帰宅ゲージ

遊びの核

プレイヤーは終電前の勇者たち。二次会ドラゴンが毎ラウンド誘い文句を吐くので、水、休憩、会計、タクシー、明日の予定カードを使って穏便に帰宅ゲージを進める。

1ラウンドの流れ

  1. ドラゴンの誘いカードを公開する。
  2. 全員が対応カードを伏せて同時公開する。
  3. 乾杯カードを使うとドラゴンの機嫌は上がるが、帰宅ゲージは止まる。
  4. 終電トラックが尽きる前に帰宅ゲージを満たせば勝利。

コンポーネント

ドラゴンカード 誘い文句カード 対応カード 帰宅ゲージ 終電トラック
サンプルカード 誘い「一杯だけだから」: 乾杯カードを出すと機嫌+2、帰宅-1。水カードを出すと機嫌+1、帰宅+1。

飲みの絡み方

飲み会あるあるを、危険飲酒ではなく帰宅ゲームに変換する。水と休憩が強いカードになる。

磨きどころ

誘い圧を肯定しすぎないこと。勝ち筋は「健全に帰る」「水を飲む」「明日を守る」に置く。

安全設計: 二次会や飲酒を強制するゲームではなく、帰宅と休憩を成功条件にする。

最初に作るならこの3つ

バカゲーは、説明が長いと急に真顔になります。最初のテストは「カードを引いた瞬間に笑えるか」「10分で終わるか」「飲まなくても同じくらい楽しいか」を見るのがよいです。

最速試作: 乾杯裁判 罪状カード30枚、証拠カード20枚、判決札だけで今日作れる。
量産しやすい: 注文してません 謎注文名と1行効果を増やすだけでリプレイ性が伸びる。
見た目が強い: 乾杯禁止区域 グラス配置と謎条例で、プレイ写真がすぐ変になる。
